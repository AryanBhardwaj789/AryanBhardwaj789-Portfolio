Book Wise

Book Wise is a dynamic and user-friendly online bookstore application developed using Java Spring Boot. This project was created as part of a Java course at Sheridan College, where I applied my knowledge of Java programming and Spring Boot framework to build a robust web application.

Key Features:
User Authentication: Secure login and registration process for users implemented using Spring Security.
Browse and Search: Users can browse through a collection of books and search for specific titles or authors.
Shopping Cart: Users can add books to their shopping cart and proceed to checkout.
Order Processing: Complete order processing system, including order confirmation and status updates.
Admin Panel: Admin users have access to an admin panel for managing books, users, and orders.


Technologies Used:
Java Spring Boot: Used for developing the backend of the application, providing a robust and scalable framework.
Thymeleaf: Used for server-side Java template engine for web and standalone environments.
HTML, CSS, JavaScript: Used for the frontend of the application to create a responsive and interactive user interface.
MySQL: Used as the database management system to store and manage book and user data.


Key Development Highlights:
Throughout the development of BookWise, I gained valuable experience in software development, including working with APIs, implementing authentication and authorization mechanisms, and ensuring code quality through testing and debugging. This project showcases my ability to develop full-stack web applications and my proficiency in Java programming and Spring Boot framework.
